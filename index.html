OpenChatTUI

Encrypted peer-to-peer terminal chat.
No servers. No logs. No traces.

Start Communicating

Launch immediately in a secure, isolated sandbox using `uv`.

uv run openchat.py

Absolute Privacy

Everything lives in RAM and dies when you close the app. Messages are end-to-end encrypted directly peer-to-peer using Military-grade AES-256-GCM and X25519 ECDH.

Beautiful Interface

A striking, high-contrast AMOLED UI built entirely in the terminal. Experience dynamic autocomplete, an intelligent escape overlay, and intuitive navigation.

Zero Configuration

No databases or central servers required. Automatic network discovery via UDP broadcast seamlessly connects you to peers locally, or connect directly via IP address.

Commands Reference

Command Description
/join <ID> Connect to a peer by Chat ID
/connect <IP:PORT> Direct connection by address
/lobby Disconnect and return to the lobby screen
/login Disconnect and return to the login screen
/save txt Export chat as plaintext file
/save html Export chat as styled HTML file
/clear Clear the screen (history preserved)
/peers List connected peers with IDs
/fingerprint Show your Ed25519 fingerprint
/verify <ID> Show a peer's fingerprint
/help List all commands
/exit Secure wipe and quit
