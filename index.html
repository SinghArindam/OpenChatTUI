Designed for Uncompromising Privacy

01

Absolute Privacy

Everything lives exclusively in system RAM and dies immediately when you exit the app. Messages are end-to-end encrypted peer-to-peer using high-grade X25519 ECDH key exchanges and AES-256-GCM encryption.

02

Stunning Interface

A high-contrast, beautiful AMOLED-friendly design crafted exclusively for modern terminals. Features fluid keyboard navigation, smart autocomplete suggestion drawers, and custom overlay menus.

03

Zero Infrastructure

No servers, central relays, or database backends. Direct host-to-host UDP broadcasts discovery resolves local network peers instantly, with optional IP routing for external nodes.

Cryptographic Architecture

OpenChatTUI is architected upon industry-standard, time-tested cryptographic primitives. Every message is secured using modern authenticated encryption, ensuring zero leakage of metadata or communication content.

Security Property Primitive Function
Key Exchange X25519 ECDH Generates ephemeral secrets without transmission
Encryption AES-256-GCM Authenticated, authenticated payload encryption
Signatures Ed25519 Out-of-band peer verification against MitM
Key Derivation HKDF-SHA256 Derives separate keys per peer pair, per direction
Forward Secrecy Ephemeral Keys Fresh session key generation on every socket connect
Data Lifecycle RAM-Only Explicit zeroing/wiping of variables upon app close

Interactive Commands Browser

Command Description
/join <ID> Connect to a peer by Chat ID
/connect <IP:PORT> Direct connection by IP and port address
/lobby Disconnect from peer and return to network lobby
/login Disconnect from lobby and return to usernames login
/save txt Export active chat history as plain text file
/save html Export active chat history as styled offline HTML
/clear Clear the chat window view (history remains intact)
/peers List active peer connections with usernames and IDs
/fingerprint Display your local cryptographic identity fingerprint
/verify <ID> Display identity fingerprint of a specified active peer
/help Display direct instructions for all terminal commands
/exit Perform variables zero-out wipe and quit interface
